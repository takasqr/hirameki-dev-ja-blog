## SQLでテーブルを作りたい

SQL Serverでテーブルを作る。GUIで作るのもいいけど、今回はSQLで作る。

`CREATE TABLE`を使ってテーブルを作る。

```sql
CREATE TABLE [dbo].[Employee]
(
    [id] [INT] PRIMARY KEY CLUSTERED,
    [name] [NVARCHAR](100) NOT NULL,
    [tel] [NVARCHAR](100) NULL
);
```
