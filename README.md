# Nuxt Layer

[hirameki.dev](https://hirameki.dev/ja) のブログ部分を Nuxt Layer として切り出したリポジトリです。

## 構成

- Nuxt3 Layer
- Tailwond
- Storybook

## Storybook

以下のコマンドで Storybook のビューアを起動できます。

```bash
npm run storybook
```

## Git 運用

- コミットメッセージは [Conventional Commits](https://www.conventionalcommits.org/ja/v1.0.0/) に従うものとします
